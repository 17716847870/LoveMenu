# LoveMenu 底部导航栏重新设计任务（购物车突出 + 悬浮导航）

## 一、设计目标

当前导航栏存在几个问题：

1. 设计过于普通
2. 购物车入口不突出
3. 导航栏紧贴底部，视觉层次不足

本任务目标是 **重新设计 BottomTabBar**，实现一个 **悬浮式情侣风导航栏**。

设计目标：

* 导航栏 **悬浮在页面上方**
* 导航栏具有 **圆角卡片设计**
* **购物车按钮突出显示**
* 支持 **四种主题风格**
* 支持 **动画效果**
* 支持 **购物车数量徽章**

组件位置：

```
/components/mobile/BottomTabBar.tsx
```

---

# 二、核心设计方案

新的导航栏采用 **悬浮式设计（Floating TabBar）**。

特点：

1 导航栏 **不贴底部**

```
bottom: 20px
```

2 导航栏 **卡片式**

```
圆角
阴影
悬浮
```

3 购物车 **中间凸起**

类似：

```
Dribbble 常见设计
Food App 导航
```

视觉结构：

```
首页    菜单     购物车     聊天     我的
        ○        ⬆
      普通      突出
```

# 五、购物车突出设计

购物车按钮需要 **明显大于其他按钮**。

视觉结构：

```
首页   菜单    ○购物车○    聊天   我的
```

购物车按钮：

```
圆形按钮
尺寸更大
悬浮在导航栏上方
```

尺寸示例：

```
普通Tab: 40px
购物车: 64px
```

CSS结构：

```
position: absolute
top: -20px
```

效果：

购物车按钮 **浮在导航栏上面**。
