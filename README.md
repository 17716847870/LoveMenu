# LoveMenu ❤️

一个为情侣设计的互动点餐系统。

> 用“亲亲”和“贴贴”作为价格，让“今天吃什么”变得更有趣。

---

## 📖 项目介绍

LoveMenu 是一个为情侣设计的小型互动点餐系统。
系统的目标是让日常生活中的“今天吃什么”变得更加有趣。

一方可以在前台浏览菜单，并选择想吃的菜。
用户可以把菜加入购物车，然后提交订单。

系统使用 **亲亲（Kiss）** 和 **贴贴（Hug）** 作为价格单位，而不是使用金钱。
这种设计可以增加情侣之间的互动感。

另一方可以在后台查看订单。
用户可以决定自己做饭，或者一起出去吃。

系统还提供聊天功能。
双方可以发送 **文字、图片、语音和表情** 进行交流。

系统的界面支持高度配置。
管理员可以修改背景图片、主题风格和情侣文案。

这个系统不仅是一个点餐工具，也是情侣之间的一种有趣互动方式。

## 💰 特殊价格系统

系统不使用金钱作为价格。

系统使用两种单位：

- **亲亲 💋**
- **贴贴 🤗**

示例：

鸡翅
价格：1 个亲亲 + 1 个贴贴

这种设计可以增加情侣之间的互动感。
